Encrypter/Decrpyter This basic file encrypter will allow basic text encryption and ecryption via the use of the node module "crypto-js". It uses AES-256 to encrypt and decrypt some message through the use of some key or password. File encryption will soon be implemented. The program is light, fast, and easy as it's basic but powerful.

File Encrypter Takes the user to another page to encrypt/decrypt an image. Similarly to regular text encryption, one needs some key or password to encrypt/decrypt an image. A new feature that's still be worked upon.
